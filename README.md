## Overview

This project was developed as part of the **DIO Power BI for Data Analytics Bootcamp**. Using Microsoft's built-in **Sample Financial** dataset, the goal was to build a comprehensive dashboard that extracts meaningful business insights through multiple visualization types and analytical perspectives.
